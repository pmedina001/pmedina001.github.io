Project Description

This capstone project aims to predict and visualize crime hotspots in Florida using AI and geospatial data. The analysis integrates socio-economic factors such as poverty rate, education levels, unemployment, and more to explore their relationship with different types of crime.

Data was collected from public sources including law enforcement crime reports and U.S. Census socio-economic data from 2015 to 2020. Machine learning models were trained to identify high-risk areas, and the results are visualized through interactive maps and charts on this website.

About the Project

This project uses AI and geospatial data to predict and visualize crime hotspots across Florida. Socio-economic indicators are factored in to analyze correlations and patterns. Created by pmedina001.
