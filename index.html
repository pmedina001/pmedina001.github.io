About the Project

This capstone project was developed by Pedro Medina as part of the MS in Data Science program with a focus in Artificial Intelligence at Florida International University. The project applies machine learning and geospatial data analysis to predict crime hotspots across Florida counties.

By integrating historical crime data with socio-economic indicators such as poverty, employment, education, and housing, the project identifies patterns that can inform proactive safety measures. The work was supervised by Dr. Levente Juhasz of Florida International University and aims to showcase how AI models, combined with spatial insights, can enhance crime prediction and policy planning.

This website provides an interactive overview of the project’s findings, visualizations, and methodology. This website outlines the project in a manner that it can be reporduced.

Project Description

Overview

This project investigates the use of artificial intelligence (AI) and geospatial data to predict and map crime hotspots across Florida. The core objective is to identify high-risk counties by analyzing historical crime records in conjunction with socio-economic indicators and spatial patterns. Through this approach, the project aims to enhance public safety insights and provide a framework for data-driven crime prevention strategies.

Data Summary

Data was collected for all 67 counties in Florida and grouped into three regional categories: North Florida (33 counties), Central Florida (12 counties), and South Florida (22 counties). The datasets span the years of 2015 to 2020.

  • Florida Department of Law Enforcement (FDLE) – Crime Data:
    • Violent Crime: Murder, Rape, Robbery, Assault
    • Property Crime: Burglary, Larceny, Motor Vehicle Theft
    • Time Range: 2015-2020
  • U.S. Census Bureau – Socio-Economic Data:
    • Median Household Income
    • Poverty Rates
    • Employment Rates
    • Educational Attainment Levels (Bachelor's degree or higher)
    • Housing Units
    • Population Density
    • Time Range: 2015-2020
  • U.S. Census Geospatial Mapping Files – Geospatial Data:
    • County-level shapefiles for Florida
    • Used to spatially align socio-economic and crime data

Methodology

The workflow consisted of data cleaning, regional grouping, exploratory data analysis (EDA), and correlation analysis to identify the most relevant predictors of crime. Selected features were then integrated within AI models to forecast crime risks as the county level. Geospatial tools were used to align data with Florida's map and build interactive risk visualizations.

Purpose of This Website

This website presents the project's findings in a clean and interactive format, allowing users to explore visualizations, data relationships, and AI-driven predictions. It also serves as a reproducible documentation of the full project for educational and professional use.

Tools & Technologies

  • R Studio: For data wrangling, statistical analysis, and correlation heatmaps
  • Leaflet.js: For creating interactive maps in the web interface
  • Tailwind CSS: For responsive styling of the website
  • GitHub Pages: For hosting and sharing the project site publicly

Exploratory Data Analysis (EDA)

Exploratory Data Analysis (EDA) was performed to uncover meaningful trends and patterns within both the crime and socio-economic datasets. The process included identifying data distributions, detecting outliers, and examining relationships across regions.

The analysis was conducted using R Studio, and visualizations were created to examine:

  • Crime rates by region (North, Central, South Florida)
  • Trends over time (2015-2020) for violent and property crimes
  • Distributions of socio-economic factors across counties
